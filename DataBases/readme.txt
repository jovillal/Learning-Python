There are 2 (sure there are more) types of data base interaction schemes, the libraries used in Python for interacting cia those schemes are:
  - For SQLite sqlite3 (native library)
  - For PostGreSQL psycopg2

To interact with dbases:
  1. Connect to the db
  2. Create a cursor object
  3. Write SQL query
  4. Commit changes
  5. Disconect
